Structure

Elements

HTML uses elements to describe the structure of web pages. HTML code is made up of characters that live inside angled brackets, these are called HTML elements. For example, these are the HTML elements (<p> </p>) that are used to create a paragraph on a web page. HTML elements are usually made up of two tags. An opening tag and a closing tag. The closing tag has a forward slash in it. For example, this is the opening tag (<p>) and closing tag (</p>) that is used to create a paragraph on a web page. Each HTML element tells the web browser something about the information that sits between its opening and closing tags.The opening and closing tags act like containers for information. Also, the character between the angled brackets indicates the tag's purpose. For example, the character p stands for paragraph which indicates it is used to create paragraphs.

The opening <html> tag indicates that anything between it and the </html> closing tag is all HTML code.

The opening <body> tag indicates that anything between it and the closing </body> tag is information that will be displayed inside the main web browser. For example, images, text, search bar, etc.

The opening <head> tag indicates that anything between it and the closing </head> tag is information about the web page. Rather than information that is shown inside the main web browser window. You will usually find the <title> element inside the <head>.

The opening <title> tag indicates that anything between it and the closing </title> tag is information that that is displayed on the top of the web browser, above where you type the URL of a website or web page that you want to vist, basically the web browser tabs.

Attributes

Attributes provide extra information about the contents of an element. They appear on the opening tag of a element and are made up of two parts. A attribute name and a attribute value which are separated by an equals sign. For example, this is a p (paragraph) element, <p lang="en-us">Paragraph in US english</p>, which contains attributes. Different attributes can have different values. The attribute name or in the case of the p element example, lang, indicates what kind of additional information you're supplying about the element's content. Also, HTML5 allows you to use uppercase attribute names but this is not recommended. Therefore, all attribute names should always be written in lowercases. The attribute value or in the case of the p element example, "en-us", is the information or setting for the attribute. It should always be placed inside double quotemarks. HTML5 allows quotemarks to be removed from attribute value. However, this is not recommended. Using the previous example, the attribute name lang, is used to indicate the language used in the p element. While, the attribute value "en-us", specifies that the p element language being used is US english.

Creating a web page Visual studio code (VS code)

To create a web page in Visual studio code, simply click on the New File button and enter your desired web page name and add to your desired web page name. The following file extension, .html. Once, you have done this, tap the enter or return key on your operating system's keyboard. This will create a HTML web page file for you. For example, creating a file called homepage.html will create and save a web page called homepage.

Headings

There are 6 levels of heading elements that can be used in HTML. These are 6 different type of HTML heading elements that can used in a HTML web page. They are:

  • <h1> </h1>
  • <h2> </h2>
  • <h3> </h3>
  • <h4> </h4>
  • <h5> </h5>
  • <h6> </h6>
The h1 element is used for main headings. So, text placed between the h1 opening and closing tag element will be a main heading. The h2 element is used for sub headings. So, text placed between the h2 opening and closing tag element will be a sub heading. The h3 to h6 elements is used for is used for even smaller headings. So, text placed between the h3 to h6 opening and closing tag elements contain smaller headings under the h1 & h2 level headings. The largest heading is the h1 heading and the smallest heading is the h6 heading. For example, this is a list of the all the headings from h1 to h6 level:

  • This is a h6 heading.
