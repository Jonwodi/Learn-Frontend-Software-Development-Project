Setup of reactjs project

In order to work with react. You first of all have to install node js onto your operating system. Node can be downloaded by using the following link to the official node website: https://nodejs.org/en/download/. Using the following link, node can be downloaded for both macOS or windowsPC users. Node can also be downloaded specifically for macOS using homebrew, if you already have homebrew on your macOS using the followwing link: https://formulae.brew.sh/formula/node Node can also be downloaded specifically for windowsPC users using chocolatey, if you already have this package on your windowsPC using the following link: https://community.chocolatey.org/packages/nodejs.

Once you have finished dowloading node onto your operating system to make sure that node was correctly dowloaded onto your system and also to identify what version of node you have on your system enter the following code, node -v, into your terminal. You then want to run the following code, npx -v, to check that you have npx and also to identify the version of npx that you have installed onto your system. Npx is simply a node package.
